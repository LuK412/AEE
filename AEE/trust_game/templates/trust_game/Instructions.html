Was in Teil 2 passiert

Jede Gruppe enthält zwei Personen: Eine Person A und eine Person B.

Person A besitzt {{ Constants.endowment }}.

Person A kann entscheiden wie viel Sie von den {{ Constants.endowment }} an Person B schicken möchte. Die Punkte, die sie an Person B schickt werden verdreifacht.

Sobald Person B die verdreifachten Punkte erhalten hat, kann sie entscheiden wie viel sie davon an Person A zurückschicken möchte.
